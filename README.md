# JsonHelper
## Description
* Visual Studio 2019, .NET Core 3.1, WPF C# application to format, encode and decode JSON.

## Features
* JSON beautify
* JSON minify
* JavaScript encode
* JavaScript decode
* Base64 encode
* Base64 decode
* GZIP compress
* GZIP decompress
* Paste from and copy to clipboard
