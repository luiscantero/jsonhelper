# JsonHelper
## Description
* Visual Studio 2022, .NET 6, WPF C# application to format, encode and decode JSON.

## Features
* JSON prettify
* JSON minify
* JavaScript encode
* JavaScript decode
* Base64 encode
* Base64 decode
* GZIP compress
* GZIP decompress
* Paste from and copy to clipboard
* Drag & drop file to input box
